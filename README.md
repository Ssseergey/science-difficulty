# science-difficulty

Это репозиторий НИР по теме Разработка метода оценки предсказуемости индивидуального потребительского поведения на основе измерения колмогоровской сложности.
В данный момент находится в разработке.

# Описание файлов

preprocess - предобработка
fill - заполнение пропущенных дней(строк) в данных нулями
main - временный файл для проверки гипотез
lzc_1d - сравнение Lempel-Ziv complexity для датафрейма с оригиналом
lzwc - сравнение Lempel-Ziv_Welch complexity для датафрейма с оригиналом
huffman - сравнение Huffman для датафрейма с оригиналом
lstm - базовая версия обучения модели предсказания
